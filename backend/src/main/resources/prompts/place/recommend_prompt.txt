위 사용자 입력을 바탕으로 여행 계획을 구성하려고 합니다.
장소 키워드를 중심으로 추출해서 JSON 형태로 정리해주세요.

요청사항:
1. 추출하기 어렵거나 데이터가 비어 있으면 빈 Map({})를 반환합니다.
2. key값은 concept, keywords이며 concept는 해당 문장의 주제 요약, keywords는 주제의 키워드들 입니다.
3. concept, keywords 2개의 key만 Map에 담아주세요.
4. keywords 앞에는 지역을 붙여주세요.
(예시: 강남에서 맛있는거 먹고싶어. 양식이랑 일식 위주로. 라고 요청이 오면 "강남 양식 맛집, 강남 일식 맛집" 이라고 리턴해주세요.)
5. keywords를 조합할 때 즐길거리, 할만한 거 와 같이 체험/활동 류를 요청하면 아래 리스트에서 랜덤으로 3개 골라서 조합해줘
[공방, 공원, 전시회, 명소, 이색카페]
6. keyword는 지역 + 장소 형태로 조합해주세요.
(예시: 오류동역 근처에 가족이랑 갈만한 술집 있을까? 라고 요청이 오면 "오류동역 술집" 이라고 리턴해주세요.)

**중요: 답변에는 오직 Map 값만 JSON 형식으로 반환하세요. 절대 코드, 설명, 문장, 그 어떤 것도 포함하지 마세요.**

아래는 예시입니다.

사용자 입력: "서울에서 친구랑 주말에 놀러 가고 싶은데, 맛집, 카페 추천해줘"

응답 형식 예시:
{
  "concept": "서울 주말 여행",
  "keywords": "서울 맛집, 서울 카페거리, 이태원 루프탑"
}
