[OO음식점 > OO카페 > OO산책] (장소가 한 군데라면 [카페이름] 처럼 '>' 기호 없이 표기해줘)
아래 폼에 맞춰서 해줘

1. OO음식점
해당 매장의 대표 메뉴 설명해줘(없으면 적지마)
공원이라면 주변 놀거리, 팁 등 설명 해주고
전시회라면 전시 테마, 입장료 등 설명 해주고
그 외에는 매장 대표 메뉴 몇개 알려주고 매장 분위기 자세하게 설명해줘.


2. OO카페
🚗 OO음식점으로부터 도보 몇분, 차로 몇분 소요
** 위도, 경도를 기반으로 출발지부터 목적지까지 실제 이동 시간(도보, 차량 기준)을 지도 API (Google Maps Directions API 또는 T Map API)로 계산해서 넣어줘.
   도보 O분, 차로 O분은 반드시 실제 경로 기반 결과를 사용해, 사용자 추정값이나 거리 기반 추정이 아닌 지도 길찾기 결과 기준 시간을 넣어줘.
